	- client/Makefile (makefile for client)



To Run Server:

Go to Server folder...
	make myftpd
	./mftpd [PORT NUM]



To Run Client:

	Go to Server Folder...
	make myftp
	./myftp [SERVER HOSTNAME] [SERVER PORT]
